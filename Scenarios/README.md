# Scenarios

This folder stores story-driven references for customer and field usage contexts.

## Suggested Subfolders

- [Customer Stories/](Customer%20Stories/)
- [Industry Use Cases/](Industry%20Use%20Cases/)
- [Field Examples/](Field%20Examples/)

## Content Guidance

- Keep scenarios focused on a real usage context.
- Add enough detail for trainers to reuse the scenario in a session.
- Use consistent tags or labels when a scenario is meant to be searched and reused.
