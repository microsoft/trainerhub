# Community-Contributions

Team member contributions that span multiple domains or are awaiting domain assignment.

## Purpose

Provide a staging area for valuable team contributions that:

- Apply across multiple domains (AI, Fabric, GitHub, Copilot)
- Are useful but don't yet fit clearly into one domain
- Need review before being assigned to a specific domain

This folder accelerates contribution velocity by removing blocking decisions about perfect categorization.

## What Belongs Here

**Included:**

- Cross-domain utilities and tools
- Contributions that apply to multiple product areas
- New asset types being evaluated for repository fit
- Quick wins and experiments shared across the team
- Content awaiting domain clarification

**Not Included:**

- Content that clearly belongs to one domain → Move to domain folder
- Broad documentation without executable components → Use SharePoint
- Deprecated or abandoned experiments → Archive or delete

## Migration Path

Community contributions should be reviewed regularly (recommended: every 90 days) to determine:

1. **Move to domain folder** - If the content is now clearly domain-specific
2. **Stay in Shared-Knowledge** - If it remains cross-domain or moves to another Shared-Knowledge subfolder
3. **Archive** - If no longer relevant or maintained

## Contribution Guidelines

- Use English as the default language for new content
- Include a README with clear purpose and scope
- Add metadata indicating which domains benefit from this asset
- Document prerequisites and validation steps
- Include author information for follow-up questions

## Review Criteria

When reviewing Community Contributions for migration:

- **Domain fit** - Does it clearly serve one domain's scope?
- **Usage pattern** - Is it being used by multiple teams or domains?
- **Maturity** - Is it stable enough to promote to a domain folder?
- **Maintenance** - Is someone actively maintaining it?

## Examples

- Multi-domain pricing calculator script (applies to AI, Fabric, Copilot)
- Cross-platform authentication helper (applies to AI, GitHub)
- Experimental troubleshooting tool being evaluated
- Contribution from team member awaiting domain review
