# Useful-Tools

Cross-domain scripts, utilities, and automation tools for trainer productivity.

## Purpose

Centralize tools that improve training delivery, asset management, or development workflows across multiple domains.

These are practical utilities that don't fit neatly into a single domain but provide value across AI, Fabric, GitHub, and Copilot work.

## What Belongs Here

**Included:**

- Cross-domain automation scripts
- Repository maintenance utilities
- Training delivery helpers (setup, cleanup, validation)
- Multi-domain diagnostic tools
- Asset management scripts
- Environment setup and configuration tools

**Not Included:**

- Domain-specific tools → Use domain Code or Skills folders
- One-time scripts without reuse value
- Deprecated or unmaintained utilities
- Product-specific SDKs or libraries → Reference official sources

## Tool Categories

Organize tools by function when adding content:

- **Setup and Configuration** - Environment preparation, prerequisites
- **Automation** - Repetitive task automation, batch operations
- **Validation** - Testing, verification, health checks
- **Diagnostics** - Troubleshooting, log analysis, debugging
- **Utilities** - General-purpose helpers, converters, formatters

## Contribution Guidelines

- Use English as the default language for new content
- Include a README for each tool with:
  - Purpose and use cases
  - Prerequisites and dependencies
  - Installation or setup instructions
  - Usage examples
  - Known limitations
- Keep tools focused and single-purpose when practical
- Document compatibility (OS, PowerShell/Bash, required tools)
- Include error handling and user-friendly output

## Quality Standards

Tools should be:

- **Reusable** - Applicable to multiple scenarios
- **Documented** - Clear purpose and usage
- **Tested** - Verified in clean environments
- **Maintained** - Updated when breaking changes occur

## Examples

- Azure resource tagging script for training environments
- Multi-domain cost report generator
- Lab environment cleanup automation
- Repository structure validator
- Training asset metadata generator
- Cross-platform authentication helper
